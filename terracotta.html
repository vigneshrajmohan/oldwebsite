Terracotta

A simple game built using MIT App Inventor

Date

July 2016

Category

Project

Status

Complete

I heard of MIT App Inventor through an instructor for a camp I went to, and I wanted to explore the ablity of the SDK. I got carried away and ended up creating a game. I integrated a global database onto the app to create a worldwide leaderboard, but I need to create a database on my own in order for me to avoid losing the information on the leader and his/her score. The current version of the app uses the shared database which is filled constantly and loses the variables I have stored.
